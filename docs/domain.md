# ドメイン知識

## 特典会とは

地下アイドルのライブ後などに行われる、ファンとアイドルの交流イベント。
ライブ会場のロビーや特設スペースで開催され、ファンはチケット（または現金）を払って参加する。

## 特典会の種別

| 種別 | 概要 |
|---|---|
| チェキ | チェキカメラで一緒に写真を撮る。最も一般的。 |
| 写メ | スマートフォンで一緒に写真を撮る。 |
| 握手 | 握手のみ。 |

> 上記以外の種別（例：動画撮影など）は特殊なケースとして発生しうるが、初期フェーズでは対象外とする。

## レギュレーションとは

特典会ごとに定められたルール。グループ・イベントによって内容が異なる。
ファンは事前にレギュレーションを確認することで、当日の特典会をスムーズに楽しめる。

### レギュレーションの主な項目

| 項目 | 必須 | 例 |
|---|---|---|
| 種別 | ✅ | チェキ、写メ、握手 |
| 料金 | ✅ | 1枚500円、2枚1000円 |
| サイン有無 | - | チェキへのサインが可/不可 |
| トーク有無 | - | 会話できる/できない |
| トーク時間 | - | 30秒、1分、制限なし |
